Our Project

Technology Stack

Our project is built with Spring Boot, using IntelliJ as our development environment. We use MySQL as our database and Spring Security with JWT for authentication and authorization. We also use Swagger for API documentation.

Authentication and Authorization

We use Spring Security with JWT for authentication and authorization, ensuring that our users' data is secure and protected.

API Documentation

We use Swagger for API documentation, making it easy for developers to understand our API and build applications on top of it.

Testing

We use automated testing to ensure that our code works as expected and to catch bugs before they become a problem. We also use GitHub for version control and testing, using pull requests and code reviews to ensure that new features and bug fixes are thoroughly tested before being merged into the main branch.
